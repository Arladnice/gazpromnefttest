# Тестовое задание Газпром нефть

## Работа с проектом

Для запуска проекта, пожалуйста, выполните следующие действия:

1. Склонировать проект:

```
git clone https://github.com/Arladnice/gazpromnefttest.git
```

2. Установить зависимости:

```
npm install
```

3. Запустить проект:

```
npm start
```
